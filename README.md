# Как запустить

### Необходимые зависимости

- Python 3.12+
- Poetry 2
- Docker/docker compose(для запуска базы данных и парсера)

---

```bash
git clone https://github.com/maniakalochka/spimex_parser.git
cd spimex_parser
docker compose up --build
```
